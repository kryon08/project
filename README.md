## Описание программы
Training Planner — это графическое (GUI) приложение для планирования и учета спортивных тренировок. Программа позволяет пользователю добавлять новые записи о занятиях, указывая дату, тип активности и её продолжительность. Все данные автоматически сохраняются в файл формата JSON, что позволяет не терять историю занятий при перезапуске приложения.

## Возможности
- Добавление записей о тренировках через удобную графическую форму.
- Строгая валидация ввода (проверка корректности формата даты и контроль, чтобы длительность была положительным числом).
- Фильтрация таблицы тренировок одновременно по двум критериям: по типу активности и по конкретной дате.
- Мгновенный сброс фильтров для возврата к полному списку.
- Автоматическое сохранение и загрузка данных через `data.json`.
- Интуитивный интерфейс с цветовым разделением кнопок (зеленая — добавление, синяя — поиск, красная — сброс).

## Инструкция по запуску
Что нужно установить на компьютер:
- Python 3.10 или выше.
- Стандартная библиотека Python (модули `tkinter`, `json`, `os` и `datetime` уже встроенные, ничего дополнительно через `pip` устанавливать не нужно).

## Запуск
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd project
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```

### Добавление тренировки
1. Заполните все поля формы:
   - **Дата (ГГГГ-ММ-ДД):** 2026-06-21
   - **Тип тренировки:** Бег
   - **Длительность(мин):** 45
2. Нажмите зеленую кнопку "Добавить тренировку"
3. Программа проверит данные, сохранит их в файл, и тренировка появится в таблице внизу окна.

### Фильтрация:
- **Применение фильтра:** В разделе "Фильтры" выберите интересующий вас тип тренировки или введите дату в поле, затем нажмите синюю кнопку "Применить".
- **Сброс фильтра:** Чтобы снова отобразить абсолютно все запланированные и пройденные тренировки, нажмите красную кнопку "Сбросить".
